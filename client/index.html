cafecito para el profe

Hice esta aplicación para implementar la API de MercadoPago en un sitio "real". 💻

El proceso es relativamente sencillo. Toda la información para la integración está disponible en la referencia de la API de MercadoPago para developers 🚀

La app fue construida con Vanilla JS y Node JS para el servidor, pero en la documentación hay data y ejemplos para otros stacks tecnológicos.

Esta app es de prueba. La motivación tras su construcción es simple: si en algún momento necesito enseñar conceptos referidos al tema, ya tendré funcionando un artefacto de software para tomar como guía. 🐱‍👤

Y aunque el fin detrás de este despliegue no es económico, si alguna vez te enseñé algo que merece la pena y querés invitarme un café... no me ofendo 😉
